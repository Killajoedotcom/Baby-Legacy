In size of 120*120.

平台规格和Voron 0一样，120*120大小。

## Motor size

Motor size：

XY motor: NEMA 14 （35步进）

Z motor: NEMA 17 （42丝杆电机）

E motor: NEMA 14 pancake （36圆饼电机）
